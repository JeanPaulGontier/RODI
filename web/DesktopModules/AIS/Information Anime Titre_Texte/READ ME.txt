	Créer 2 modules : 
		Admin Information Anime Titre_Texte
		Information Anime Titre_Texte
	Créer une page PORTANT LE NOM "Admin Information Anime Titre_Texte" avec le module Admin Information Anime Titre_Texte

	Si c'est pas déjà fait,
	Il faut aussi importer le fichier C:\Dossiers\RODI\src\web\Portals\_default\Skins\Rodi2017\popUpSkin.ascx (pour le jquery fonctionne)
